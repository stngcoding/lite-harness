# Issue Tracker: GitHub Issues

## Overview

Issues for this project are tracked in [GitHub Issues](https://github.com/stngcoding/lite-harness/issues).

## How skills interact with this repo

Engineering skills (`to-issues`, `triage`, `to-prd`) use the GitHub CLI (`gh`) to:
- Read existing issues
- Create new issues
- Apply labels and update issue state

## Workflow

1. **Create issues** — `gh issue create` (or via GitHub web UI)
2. **Triage** — apply labels from `docs/agents/triage-labels.md`
3. **Ready for work** — agent picks up when labeled `ready-for-agent`

See `docs/agents/triage-labels.md` for the label vocabulary.
